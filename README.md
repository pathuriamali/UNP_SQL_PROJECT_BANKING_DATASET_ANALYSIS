# UNP_SQL_PROJECT_BANKING_DATASET_ANALYSIS
Description This project demonstrates a Banking Database Management System using MySQL. It includes the creation of a complete banking schema with customers, branches, accounts, and transactions tables. The database structure uses primary keys, foreign keys, constraints, and relational design to maintain data integrity. Sample data is inserted to simulate real banking operations such as account creation and financial transactions. Various SQL queries are implemented to retrieve and analyze banking data efficiently. The project covers important SQL concepts including SELECT, WHERE, AND/OR, IN, BETWEEN, LIKE, and ORDER BY. Aggregate functions such as COUNT, SUM, AVG, MIN, and MAX are used for data analysis. Advanced SQL operations like GROUP BY, HAVING, JOINs (INNER, LEFT, RIGHT), and UNION are also demonstrated. The database is further modified using ALTER, UPDATE, INSERT, and DELETE operations. This project helps understand practical database design and query optimization in banking systems.
